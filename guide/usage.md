# Использование приложения

## Главный экран

<u>Основные элементы интерфейса:</u>

- **Кнопка перехода** - открывает экран обработки касаний
- **Область рисования** - отображает графические фигуры
- **Информационная панель** - показывает текущий масштаб

## Работа с графикой

// Пример кода рисования фигур

    p.setStrokeWidth(10);
    p.setColor(Color.RED);
