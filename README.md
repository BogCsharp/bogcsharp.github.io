# Мобильное приложение LR61

**Документация по приложению для работы с графикой и жестами**

---

##Основные возможности

- **Рисование графических фигур** (пятиугольники, шестиугольники)
- **Обработка мультитач жестов** 
- **Масштабирование изображения** двумя пальцами
- **Визуализация данных** о касаниях

<u>Приложение демонстрирует передовые технологии обработки пользовательского ввода в Android</u>
